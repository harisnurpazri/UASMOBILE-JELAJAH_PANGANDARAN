Tambahkan file font Noto ke folder ini agar Flutter dapat menampilkan karakter yang hilang.

Rekomendasi file (unduh dari Google Noto Fonts atau sumber terpercaya):
- NotoSans-Regular.ttf
- NotoSans-Bold.ttf

Setelah menaruh file ke folder ini, jalankan:

flutter pub get

Kemudian jalankan aplikasi untuk memastikan font berhasil diload.
